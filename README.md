# SimpleSales Website

A clean and simple sales website built using only HTML and CSS.

## Project Overview
This project is a basic front-end sales website designed to showcase products in a simple and organized layout.  
It focuses on structure.

## Technologies Used
- HTML5
- CSS3

## Features
- Responsive layout
- Product display section
- Clean and modern design
- Organized page structure
